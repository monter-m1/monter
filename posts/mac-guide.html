How to run piracy apps & plugins on Mac OS

Apple controls its entire ecosystem. On the Mac, even if you don't download an app from the AppStore, but from a developer's site. To run th app on Macs, the developer must get a digital signature from Apple, otherwise it will be impossible to launch the app in the usual way.

Every time you launch a new app on your Mac, Gatekeeper checks to see if it is signed with a valid signature. If the application is signed with a valid signature, it can be started. If this is not the case, you will see a warning message and your Mac will not allow you to launch the app.

To able to run piracy apps, you need to disable Gatekeeper. For it open Termianl app, then copy & paste a command below, then tap Enter

sudo spctl --master-disable

But with each new version of the Mac OS, disabling Gatekeeper helps less and less. But there is a way out — you can disable protection for a specific apps and plugins by entering the following command in the terminal:

sudo xattr -rd com.apple.quarantine /Applications/LockedApp.app
